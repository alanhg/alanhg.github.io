so,这篇文章应运而生，规则如下。欢迎评论交流。

![](http://static.1991421.cn/2019-09-27-141209.jpg)

## 宗旨

- as short as possible
- easy to read
- user-friendly input when the user enters the URL
 
## 例子

### User list
```
/users

```
### User detail
```r
/user/:id

```

### Add user

```
/user/new

```

### User's functional page

```
/user/:id/tel

```
